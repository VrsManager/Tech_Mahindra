This repository contains automated UI tests for the Parabank demo application.The project was built as part 
of an SDET take‑home assignment to demonstrate test strategy, automation design, and quality engineering practices.

Functional flows covered:

Login

Registration

Products

Focus areas: authentication, validation, and negative testing.


Project Structure:
├── src/
│   ├── Pages/              # Page Object classes
│   ├── Tests/              # NUnit/xUnit test classes
│   ├── TestData/           # JSON test data files
│   └── Utils/              # Helpers (WaitHelper, DriverFactory, etc.)
├── docs/
│   └── TestStrategy.md     # Test strategy document under Testdata
├── README.md
└── Utilities
